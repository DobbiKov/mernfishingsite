# MERN FISHING SITE

Developer: [**Roman Zapotockiy**](https://t.me/dobbi_crmp) (***dobbikov***)

## Stack
  - BackEnd
    - Node JS
    - Express
    - MySQL
  - FrontEnd
    - React JS
    - React Router DOM
    - Fetch
## Setup
1. Install the following:
  - [Node JS](https://nodejs.org)
  - [React JS](npm install react)
2. **npm install**
3. **npm run server && npm run client**
